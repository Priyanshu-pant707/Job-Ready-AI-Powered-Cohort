
# Important array function

- push() :  add at end
- pop() : remove from end
- shift() : remove from last
- unshift() : add at start
- indexOf() : return index or -1
- filter() :  return elements that satisfy condition.
```
let nums = [1, 2, 3, 4];
let even = nums.filter(n => n % 2 === 0);

```


- some() : return true is any element satisy conditions
- map() : Trnaforms each element
```
let sq = nums.map(n => n * n);

```
- reduce() : used to accumulate the values
